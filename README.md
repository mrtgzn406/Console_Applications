# Açýklama

Bu repository'de ben hangi programlama dillerine hakim olduðumu, hangilerinin sözdizimine aþina olduðumu 
gösteren konsol uygulamalarý yaptým.

Özellikle bu konsol uygulamasý yaptýðým dillerden herhangi birisinin nesne tabanlý programlama (OOP)
özelliði varsa o özelliði kullanarak karmaþýk konsol uygulamalarý geliþtirdim. Çünkü bir dilin OOP özelliðini
kullanarak konsol uygulamasý yazmak bu dillerin sözdizimine ne kadar hakim  olduðumu ispatlayan bir emaredir.
